MadeForYou

Surprise Pages

Core idea

Turn a small question into a personal mini web page.

Add your name, the other person's name, a question, optional date, time, place, and image details, then choose a visual theme and share the finished page as a single link.

Features

Everything needed for a sweet, playful surprise.

Personal page builder

Create a page with names, a question, optional details, and a custom image.

Themed experiences

Choose from different visual themes with soft animation and a mobile-friendly layout.

Playful buttons

Add fun Yes and No button behavior, including moving or changing No buttons.

Shareable link

Turn the surprise into one simple link that can be sent to the other person.

Memory download

After the answer, download the moment as a PNG keepsake.

English and Turkish

Build and share surprise pages in Turkish or English.

Use cases

Made for romantic invitations, birthdays, apologies, and small moments.

MadeForYou can be used for date invitations, anniversary pages, birthday surprises, apology messages, prom-style questions, or any small moment that feels better as a personal page than a plain message.

How it works

A short creation flow becomes one link.

The user enters the basic details, picks a theme, chooses the button behavior, and generates a private surprise page. When the receiver opens the link, they see the custom page and can answer. After the answer, the creator can save the memory as a PNG.

Status

MadeForYou is currently being built.
